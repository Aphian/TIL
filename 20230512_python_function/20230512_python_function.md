## 함수 사용 이유
- DRY (Don't repeat yourself)
  - 같은 기능을 구현하기 위해 반복적으로 같은 코드를 쓰지 않을 수 있따.
- Easy To Read
  - (함수 이름을 잘 짓는다면) 전체 코드의 가독성이 좋아진다
- Maintain
  - 프로그램에 문제가 발생하거나 기능의 변경이 필요할 때에도 손쉽게 유지보수 가능
