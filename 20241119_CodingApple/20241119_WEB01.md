### WEB_01

#### Coding과 HTML
- 사람이 하는일 -> 원인 -> `Code`
- 원천 -> `Source`
- 약속 -> 컴퓨터 언어
- 기계가 하는 일 -> 애플리케이션, 앱, 응용 프로그램, 프로그램
- 웹에서는 결과 -> 웹 페이지 -> 모이면 웹 사이트
- 기능들이 더 쌓이면 웹 애플리케이션


**원인인 코드를 통해서 결과를 만든다는 것 -> 코드를 이해하는 핵심**

- 웹 페이지를 만드는 코드 -> `HTML`
- `HTML` : 어떠한 언어보다 쉽다, 인류가 생산하는 거의 모든 디지털 정보가 담기는 그릇
- `Public Domain` : 저작권이 없는 것들
