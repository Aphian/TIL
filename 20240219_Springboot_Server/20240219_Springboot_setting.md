### Springboot Setting

- `JDK (Java Development Kit)` 설치
- `STS (Spring Tool Suite)` 설치
- `Create new Spring Starter Project` 시작
- `STS` 안에서 `JDK` 오류 해결을 위한 설정
  1. `File -> Properties`
  2. `Java Build Path -> Libraries` 오류 표시 `Edit` 클릭
  3. `Alternate JRE -> Installed JREs`
  4. `ADD -> Standard VM`
  5. `JDK` 폴더 경로 지정 후 적용
