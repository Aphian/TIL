### React_Spring_21

#### React (API Server 통신)
- 대부분의 프론트엔드는 비동기 통신을 필요로함
- `React`도 서버 데이터를 `Ajax`를 활용 하여 기능 구현
- `Springboot`를 활용하여 서버 사이드 데이터 처리

#### React (Ajax)
- `npm install axios` 명령어 실행 -> `Axios` 라이브러리 추가
- `axios`는 기본이 `json`처리이기 때문에 다루기가 쉬워짐.
- 따로 빼놓고 개발하는게 유지보수에 좋음.
