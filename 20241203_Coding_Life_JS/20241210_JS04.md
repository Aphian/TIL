### JavaScript_04

#### JavaScript 조건문
- `Boolean` 의 코드가 `True` 면 실행 / `False` 면 실행하지 않음.

#### JavaScript 문법
- `document.querySelector('#id').value` --> `id`를 가지고 있는 태그에 값을 가져옴.

#### Refactoring
- 코드에 비효율적인 부분을 효율적으로 만들어 주는 기법
- 중복 제거
- 가독성을 높여주는 방법
- `this` 키워드를 활용하여 자기 자신을 가르키는 코드로 작성 가능
- 변수라는 기능을 이용하여 중복 제거 가능
